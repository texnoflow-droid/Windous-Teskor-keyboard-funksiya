## Shift bilan ishlaydigan foydali tugmalar

| Tugmalar                    | Vazifasi                                    |
| --------------------------- | ------------------------------------------- |
| Shift + Delete              | Faylni butunlay o‘chirish (Trash'ga emas)   |
| Shift + F10                 | O‘ng tugma menyusini ochish (Mouse o‘rniga) |
| Ctrl + Shift + Esc          | Bevosita Task Manager’ni ochadi             |
| Shift + Click               | Dasturlarni boshqa oyna sifatida ochish     |
| Shift + Alt + PrintScreen   | Yordamchi kirish sozlamalarini ochadi       |
