## 🪟 Windows (Win) Tugmasi
Eng ko'p ishlatiladigan "tizimli" qisqa yo'llar:

| Tugmalar | Vazifasi |
| :--- | :--- |
| `Win` | "Start" menyusini ochish/yopish. |
| `Win + D` | Barcha oynalarni yashirish (Desktopga o'tish). |
| `Win + E` | "File Explorer" (Provodnik)ni ochish. |
| `Win + L` | Kompyuterni qulflash (Lock). |
| `Win + V` | Bufer (Clipboard) tarixini ko'rish. |

---

## ⌨️ Ctrl Kombinatsiyalari
Matn va fayllar bilan ishlash uchun asosiy vositalar:

| Tugmalar | Vazifasi |
| :--- | :--- |
| `Ctrl + C` / `V` | Nusxa olish va joylashtirish. |
| `Ctrl + X` | Kesib olish (Cut). |
| `Ctrl + Z` | Oxirgi amalni bekor qilish (Undo). |
| `Ctrl + Shift + Esc` | "Task Manager"ni to'g'ridan-to'g'ri ochish. |
| `Ctrl + T` | Brauzerda yangi tab ochish. |

---

## 🔄 Alt va Shift Tugmalari
Oynalar va tizim amallari uchun:

| Tugmalar | Vazifasi |
| :--- | :--- |
| `Alt + Tab` | Ochiq dasturlar orasida tezkor almashish. |
| `Alt + F4` | Faol oynani yoki dasturni yopish. |
| `Shift + Delete` | Faylni savatga tashlamasdan butunlay o'chirish. |
| `Alt + Enter` | Tanlangan faylning xususiyatlarini (Properties) ko'rish. |

---

## 📑 F Tugmalari (F1–F12)
| Tugma | Vazifasi |
| :--- | :--- |
| `F2` | Tanlangan fayl yoki papka nomini o'zgartirish. |
| `F5` | Sahifani yoki oynani yangilash (Refresh). |
| `F11` | Brauzerda to'liq ekran rejimiga o'tish. |
