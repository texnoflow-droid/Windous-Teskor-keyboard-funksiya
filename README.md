## Windous – Tezkor klaviatura funksiyalari

Bu loyiha Windows operatsion tizimidagi eng ko‘p ishlatiladigan **tezkor tugmalar (keyboard shortcuts)** to‘plamidir.  
Maqsad – yangi foydalanuvchilarga ham, tajribali foydalanuvchilarga ham kompyuterdan tez va samarali foydalanishda yordam berish.

### Nimalar bor?

- **Windows (Win) tugmasi bilan qisqa tugmalar**  
  `win-shortcuts.md` faylida Windows tugmasi bilan ishlaydigan barcha asosiy kombinatsiyalar (Win + D, Win + E, Win + L va hokazo) jadval ko‘rinishida keltirilgan.

- **Ctrl bilan ishlaydigan mashhur tugmalar**  
  `ctrl-shortcuts.md` faylida nusxa olish, qo‘yish, bekor qilish, tablar bilan ishlash kabi eng ko‘p ishlatiladigan `Ctrl` kombinatsiyalari jamlangan.

- **Alt bilan ishlaydigan tugmalar**  
  `alt-shortcuts.md` faylida oynalar orasida almashish, oyna menyusi, dastur oynasini yopish kabi `Alt` tugmasi bilan ishlaydigan kombinatsiyalar bor.

- **Shift bilan foydali tugmalar**  
  `shift-shortcuts.md` faylida faylni butunlay o‘chirish, o‘ng tugma menyusini klaviaturadan chaqirish, Task Manager’ni tez ochish va boshqa foydali kombinatsiyalar jamlangan.

- **F tugmalari (F1–F12)**  
  `f-keys-shortcuts.md` faylida yordam oynasi, fayl nomini o‘zgartirish, sahifani yangilash, to‘liq ekran rejimi kabi funksiyalar bilan bog‘liq `F` tugmalari keltirilgan.

### Qanday foydalanish mumkin?

- Fayllarni ochib, kerakli bo‘lim bo‘yicha tezkor tugmalarni o‘rganing.
- Darslar, prezentatsiyalar yoki kompyuter savodxonligi bo‘yicha kurslarda qo‘llanma sifatida foydalanishingiz mumkin.
- O‘zingizga kerak bo‘lgan qisqa tugmalarni belgilab, oson eslab qolish uchun chop etib olishingiz mumkin.

### Hissa qo‘shish

Agar sizda qo‘shimcha foydali tugmalar yoki takliflar bo‘lsa:

- yangi branch oching,
- o‘zgartirishlarni kiriting,
- `pull request` yuboring.

Birgalikda bu qo‘llanmani yanada boy va foydali qilamiz.
